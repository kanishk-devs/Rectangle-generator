# Rectangle generator
A simple C++ program to generate rectangles using loops.

## Features
- Takes user input for height and width
- Prints a hollow rectangle pattern
- Uses nested loops (for loops)

## Tech Used
- C++

## How to Run

1. Compile the code:
   g++ rectangle.cpp -o rectangle

2. Run the program:
   ./rectangle
